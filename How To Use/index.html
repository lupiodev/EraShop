Attributes

  • In this section admin can view current attribute , add new attribute with edit and delete options.
  • Overall, the attributes page description helps you understand the different characteristics (attributes) used to classify products or items in the EraShop system.

Category & Sub Category

  • The category page description helps you understand the list of category. By using the filters, buttons, and other information provided, you can efficiently add , view , delete and modify category information.
  • Click info and open sub category page.
  • The sub category page description helps you understand the list of category wise sub category. By using the filters, buttons, and other information provided, you can efficiently add , view , delete and modify sub category information.

Promocode

  • In this promocode section ,The EraShop promocode allows you to add, update, view, and delete promocode information..

product request

  • In this pending product request section , The EraShop pending product request section allows accept and decline sellers to submit requests for new products and update products..
  • The EraShop pending product request seems to be a tool for viewing and searching pending product request.
  • In this approved product request section , The EraShop approved product request section view and allows search for new products and update products..
  • In this rejected product request section , The EraShop rejected product request section view and allows search for new products and update products..

Real Or Fake products

  • In this section a real product page functionality allows to view , modify and delete with searching for a real product.
  • Admin can also handle new collection and out of stock switch for a real products.
  • In this section a fake product page functionality allows to add , view , modify and delete with searching for a fake product.
  • Admin can also handle new collection and out of stock switch for a fake products.
  • Click on the "Info" button (i) to view a product detail page with more details about the product without leaving the current list.

Real Or Fake Reels

  • In this section a real reels page functionality allows to view , modify and delete with searching for a real reels.
  • In this section a fake reels page functionality allows to add , view , modify and delete with searching for a fake reels.
  • Click on the "Info" button (i) to view a reel detail page with likes , commets and more details about the reels without leaving the current list.

Order

  • In this section a order page functionality allows modify order status with searching for a order.
  • When click on orderId it redirect to order detail page with order amount , discount , promocode and delivery address etc...

Live Seller

  • In this section a live seller page functionality admin can view all live seller
  • However, the ability to view seller products seems to be restricted. You can only view products if the seller is designated as a "Real Live Seller" in the system.
  • When click on view product it redirect to seller products page.

Seller

  • In this section seller request page functionality admin can view and modify seller request with searching to seller request.
  • The EraShop real seller management allows you to view a list of real seller, and allows modify, filter and search for them, and potentially access detailed information about individual sellers.
  • In this section real seller when click notification icon and allows to direct send notification to personal real seller.
  • When click on info button and it redirect to seller detail page with seller personal detail, bank detail , order , products , transaction etc...
  • The EraShop fake seller management allows you to view a list of fake seller, and allows to add , modify, delete , filter and search for fake seller and handle isLive switch them.

Users

  • The EraShop user management allows you to view a list of all users and allows to block/unblock users.
  • When click on info button and it redirect to user detail page with user personal detail and orders .

Seller Withdraw

  • In this section admin can handle seller withdraw request and pay them.

Admin Wallet

  • The EraShop admin wallet allows you to view a admin earning , earning with commission and daily , monthly and yearly earning with date filter.

  • Also admin can handle razor pay , stripe pay switch
